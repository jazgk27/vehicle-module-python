# Vehicle Module Analysis
Class project analyzing vehicle characteristics using a custom Python module.

## Description
This script creates and uses a Python module that evaluates vehicle reliability, fuel efficiency, and resale value based on user input.
The program performs basic conditional checks, applies calculations, and outputs a simple report with the results.

## Technologies Used
- Python 3

## How to Run the Project
1. Place both files in the same folder:
   - `vehicle.py`
   - `main.py`

2. Run the program:
   python main.py

3. Enter the requested values when prompted:
   - Vehicle age
   - MPG rating
   - Original cost

## Main Features

- Determines reliability based on vehicle age
- Evaluates fuel efficiency (MPG ≥ 30)
- Calculates resale value using 3% annual depreciation
- Displays formatted output with results

## Skills Demonstrated

- Creating and using Python modules
- Writing functions with parameters
- Using conditional statements
- Performing basic calculations
